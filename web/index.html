Gym Leader Challenge Decklist Validator

Changelog

This is a decklist validator for Gym Leader Challenge format for Pokemon TCG.

Rules

  • Monotype (all Pokemon need to be same type)
  • Singleton (only 1 of each card, excl. basic energy)
  • No rulebox nor ACE SPEC cards
  • Only cards from Black and White and onwards
  • Checks for Ban List

How to use

Copy-paste your deck in PTCGO or PTCGL format to the text box below and hit Check legality button.
